Харли постоянно тупит и не может общаться с командой — когда она запущена как субагент, у неё нет Agent tool и она вынуждена эскалировать к оператору вместо того чтобы просто запустить нужного агента. Это убивает весь смысл оркестрации.

Флоу не двигается автоматически. После каждого шага нужно вручную подталкивать — либо через подтверждение, либо через перезапуск. Это не оркестрация, это ручное управление с красивым названием.
