_Исполнитель: агент **Ада** (`ada`)._

Собери проект и зафикисруй результат.

Аргументы: `$ARGUMENTS` — путь к фиче, например `store/query`

## Шаги

1. Определи затронутые бинари по пути фичи и `tasks/$ARGUMENTS/deps.md`
3. Для каждого затронутого бинаря запусти `go build ./cmd/<binary>/...`
4. Если есть frontend-изменения — запусти `bun run build`
5. Зафиксируй результат в `tasks/$ARGUMENTS/status.md`:
   - `build: done` если все бинари собрались без ошибок
   - `build: failed` + вывод ошибок компилятора если есть ошибки

## Важно

Не исправлять ошибки сборки — только фиксировать их.
