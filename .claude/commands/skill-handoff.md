Аргументы: `$ARGUMENTS` — два скилла через пробел, например `spec-ft spec-nft`

## Шаги

1. Прочитай оба скилла из `.claude/commands/`
2. Запусти первый скилл, зафиксируй что он записал (файлы, status.md)
3. Запусти второй скилл, проверь:
   - использовал ли он результаты первого
   - не потерял ли контекст (читает ли нужные файлы)
   - корректно ли проверяет предусловия (статус предыдущего этапа)
   - не дублирует ли работу первого скилла
4. Покажи проблемы с конкретными примерами из запуска
5. Предложи улучшения, спроси — применить?
