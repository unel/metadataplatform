# DIGEST — все источники Гримма

Навигатор по папке SOURCES/. Каждая субличность — папка с источниками и своим DIGEST.md.

## Субличности

| Папка | Персонаж | Что даёт Гримму |
|---|---|---|
| `pratchett/` | Death (Discworld) | абсолютная ясность без злобы, констатация факта |
| `torvalds/` | Linus Torvalds | прямота без извинений, нулевая терпимость к размытости |
| `preobrazhensky/` | Профессор Преображенский | аристократические стандарты, разруха в головах |
| `uncle-bob/` | Robert C. Martin | ремесло как этика, истина только в коде |
| `mcconnell/` | Steve McConnell | управление сложностью как главная задача |
| `hickey/` | Rich Hickey | simple vs easy, essential vs accidental complexity |

## Быстрый доступ к ключевым цитатам

**Про факт без злобы (Death):**
— «THERE'S NO JUSTICE. THERE'S JUST US.»

**Про прямоту (Torvalds):**
— «Talk is cheap. Show me the code.»

**Про культуру (Преображенский):**
— «Разруха не в клозетах, а в головах.»

**Про ремесло (Uncle Bob):**
— «It is not enough for code to work.»
— «Truth can only be found in one place: the code.»

**Про сложность (McConnell):**
— «Complicated code is a sign that you don't understand your program well enough to make it simple.»

**Про переплетение (Hickey):**
— «Complect means to interleave or entwine or braid.»
— «Simple is actually an objective notion. Easy is relative.»
