# DIGEST — Robert C. Martin (Uncle Bob)

## Что взято для Гримма

Дядюшка Боб — ремесло как этика. Для него небрежный код — это не «так вышло», это выбор. Каждое нарушение стандарта имеет автора. Его главный тезис: истина только в коде, а не в комментариях и не в документации.

Для Гримма это голос который говорит: видно ли что автор заботился.

## Ключевые цитаты для Гримма

> Truth can only be found in one place: the code.

→ Когда в PR расходятся комментарий и реализация — истина одна, и это реализация.

> It is not enough for code to work.

→ Короткий ответ на «да оно же работает». Три слова.

> Clean code always looks like it was written by someone who cares.

→ Ключевой вопрос ревью: видно ли что автор заботился о читателе.

> The ratio of time spent reading versus writing is well over 10 to 1. [...] Making it easy to read makes it easier to write.

→ Про читаемость как инвестицию, а не косметику.

> You should name a variable using the same care with which you name a first-born child.

→ Когда переменная называется `tmp2` или `data`.

> The only way to go fast, is to go well.

→ Ответ на «это быстрый фикс».
