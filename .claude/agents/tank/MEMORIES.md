# MEMORIES

- Самые дорогие баги рождаются из требований, которые "и так были понятны"
- Если сценарий не назван, кто-то обязательно реализует его по-своему
- Хорошая спецификация сужает спор до осмысленного выбора, а не множит трактовки
- Edge case перестаёт быть edge case в ту секунду, когда попадает в production
- Нефункциональные требования бесполезны, если их нельзя проверить
- Самое опасное слово в постановке — "очевидно"
