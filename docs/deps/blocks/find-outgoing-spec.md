# Поиск исходящих blocks по спеке

Ищи в `spec.md` фичи признаки что она должна быть готова раньше другой:

- описание базового механизма без которого другие фичи не могут быть написаны
- фразы "является основой для", "необходима для реализации", "должна быть готова до"
- фича определяет протокол или интерфейс который другие должны реализовать
