# Поиск исходящих blocks по коду

## Go
- пакет определяет интерфейс который другие пакеты должны реализовать
- пакет инициализирует разделяемый ресурс (пул соединений, сокет) используемый другими

## SvelteKit

### lib/types/
- типы которые должны быть определены до того как другие модули могут быть написаны

### lib/stores/
- стор который инициализирует глобальное состояние необходимое другим сторам

## CSS
- файл с design tokens который должен быть определён до компонентных стилей
