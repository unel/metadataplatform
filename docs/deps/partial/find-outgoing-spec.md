# Поиск partial по спеке (исходящие)

Ищи в `spec.md` фичи признаки что часть её функциональности уже есть в другой:

- описание поведения которое звучит как подмножество другой фичи
- фразы "частично", "в том числе", "среди прочего делает X" где X — уже есть в другой фиче
- одинаковые операции над одними и теми же данными но в разном контексте
