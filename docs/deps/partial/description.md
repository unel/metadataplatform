# partial

Часть функциональности A уже реализована в B.

**Когда использовать:** при написании A обнаружено что B уже делает часть этого.
Стоит решить — вынести общее, переиспользовать, или это нормальное разделение.
**Обратная связь:** симметричная
**Пример:** `platform/job partial store/upsert` — platform частично повторяет логику upsert

**Не путай с:**
- `overlaps` — перекрытие на уровне интерфейса, не реализации
- `extends` — A намеренно строится поверх B
