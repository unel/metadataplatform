# Поиск partial по коду (исходящие)

## Go
- функции которые повторяют логику из другого пакета но не импортируют его
- одинаковые SQL запросы в разных пакетах
- дублирующаяся валидация или трансформация данных

## SvelteKit

### lib/components/
- одинаковая логика в разных компонентах которая могла бы быть вынесена в `$lib`

### lib/stores/
- повторяющаяся логика трансформации данных в разных сторах

### lib/api/
- дублирующаяся обработка ответов API в разных модулях

## CSS
- повторяющиеся наборы свойств которые могли бы стать utility классом или миксином
