# Поиск исходящих extends по спеке

Ищи в `spec.md` фичи признаки что она строится поверх другой:

- фразы "расширяет", "специализирует", "является частным случаем", "добавляет к"
- описание поведения которое повторяет другую фичу но с дополнениями
- явные ссылки на базовую фичу как на основу
- переопределение части поведения базовой фичи
