# Поиск conflicts по спеке (исходящие)

Ищи в `spec.md` фичи признаки что она противоречит другой:

- требования которые явно исключают поведение описанное в другой фиче
- одинаковые ресурсы (сокет, таблица, файл) с разными ограничениями доступа
- НФТ которые несовместимы — например одна фича требует эксклюзивного доступа, другая — разделённого
- разные ожидаемые форматы одних и тех же данных
