# Поиск исходящих used-by по спеке

Ищи в `spec.md` фичи признаки что она предоставляет что-то другим:

- фразы "предоставляет", "экспортирует", "используется как", "является источником"
- описание интерфейса или протокола который другие компоненты должны соблюдать
- упоминание того что другие фичи читают/пишут через эту
