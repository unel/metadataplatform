# used-by

B используется в A — обратная связь к `depends-on`.

**Когда использовать:** проставляется автоматически когда другая фича
декларирует `depends-on` на эту.
**Обратная связь:** `depends-on`
**Пример:** `store/query used-by api/entities` — store используется в api.

**Не путай с:**
- `extends` — B не просто использует A, а строится поверх неё
