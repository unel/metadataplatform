# Поиск входящих depends-on по коду

## Go

- grep по имени пакета этой фичи в импортах других пакетов
- grep по имени сокета этой фичи (`store.sock`, `spawner.sock` и т.п.)
- grep по именам типов и интерфейсов которые экспортирует эта фича
- grep по путям сокетов в YAML конфигах других компонентов

## SvelteKit

### routes/
- grep по имени API endpoint этой фичи в `fetch()` вызовах других роутов
- grep по имени param сегмента если эта фича определяет структуру роута

### lib/components/
- grep по имени компонента в импортах других компонентов и роутов
- grep по имени prop типа который экспортирует этот компонент

### lib/stores/
- grep по имени стора в импортах других сторов, компонентов и роутов
- grep по имени стора в `derived()` вызовах

### lib/api/
- grep по имени функции API клиента в роутах и компонентах
- grep по имени типа ответа который экспортирует этот модуль

### lib/types/
- grep по имени типа в импортах других модулей

## CSS

- grep по имени CSS custom property (`--token-name`) в других CSS файлах и `.svelte` компонентах
- grep по имени utility класса в других компонентах
- grep по пути `@import` в других CSS файлах
