# Поиск входящих depends-on по спекам

Ищи в `spec.md` других фич признаки что они зависят от этой:

- прямые упоминания имени этой фичи или её компонента
- фразы "читает из X", "пишет в X", "использует X" где X — эта фича
- описание входных данных совпадающее с выходными данными этой фичи
- упоминание сокета или интерфейса который предоставляет эта фича
