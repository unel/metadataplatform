# Поиск исходящих depends-on по спеке

Ищи в `spec.md` фичи признаки что она использует другой компонент:

**Прямые упоминания:**
- названия других фич или бинарей (`store`, `spawner`, `api`, `platform`)
- названия сокетов (`store.sock`, `spawner.sock`)
- фразы: "читает через", "пишет в", "отправляет команду", "вызывает", "использует"

**Косвенные признаки:**
- упоминание формата данных который определён в другой фиче (например JSONL-протокол)
- ссылки на типы или структуры из другого компонента
- описание входных данных которые приходят из другой фичи
