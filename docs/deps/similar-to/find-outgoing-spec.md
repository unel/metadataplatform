# Поиск similar-to по спеке (исходящие)

- одинаковый паттерн взаимодействия (поллинг, spawn, JSONL и т.п.)
- похожая структура входных/выходных данных но для разных сущностей
- аналогичный жизненный цикл или state machine
- фразы "по аналогии с", "так же как", "аналогичный механизм"
