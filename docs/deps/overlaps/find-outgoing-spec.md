# Поиск overlaps по спеке (исходящие)

Ищи в `spec.md` фичи признаки что она делает то же что другая:

- описание поведения которое звучит как уже описанное в другой фиче
- одинаковые входные/выходные данные у разных фич
- фразы "так же как", "аналогично", "похожим образом" со ссылкой на другую фичу
- две фичи работают с одними и теми же сущностями одинаковым способом
