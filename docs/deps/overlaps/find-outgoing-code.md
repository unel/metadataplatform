# Поиск overlaps по коду (исходящие)

## Go
- одинаковые SQL запросы или одинаковая бизнес-логика в разных пакетах
- дублирующиеся типы или структуры с одинаковыми полями
- одинаковая обработка ошибок или валидация в разных местах

## SvelteKit

### lib/components/
- компоненты с одинаковой разметкой и логикой но разными именами
- одинаковые вычисления в нескольких компонентах

### lib/stores/
- сторы которые хранят одни и те же данные
- одинаковая логика derived сторов

### lib/api/
- дублирующиеся fetch вызовы к одним и тем же endpoints

## CSS
- одинаковые наборы свойств в разных классах
- дублирующиеся значения CSS custom properties
