# Поиск исходящих blocked-by по коду

## Go
- импорты пакетов которые ещё не существуют (compile error)
- обращения к сокетам или ресурсам которые инициализируются в другом пакете

## SvelteKit
- импорты типов или компонентов которые ещё не написаны
- использование stores или API функций которые ещё не определены

## CSS
- использование CSS custom properties которые ещё не определены
