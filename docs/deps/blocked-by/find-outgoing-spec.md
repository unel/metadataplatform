# Поиск исходящих blocked-by по спеке

Ищи в `spec.md` фичи признаки что она зависит от готовности другой:

- фразы "требует", "предполагает наличие", "не может работать без"
- описание предусловий которые должны быть выполнены до начала реализации
- ссылки на базовый механизм или инфраструктуру которая ещё не готова
