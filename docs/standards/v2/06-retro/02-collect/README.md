# 06-retro/02-collect: Сбор выжимок

## Что делает

Харли создаёт retro-команду, спаунит всех участников фичи параллельно. Каждый читает свои notes и complaints, формирует выжимку (3–5 пунктов) и отправляет Харли. Харли собирает список проблем с атрибуцией.

## Входящие артефакты

- `06-retro/01-recall/report-NNN.md` — список участников
- `tasks/<feature>/notes-<агент>.md` и `complaints-<агент>.md` — заполненные на шаге recall
- `tasks/<feature>/notes-user.md` и `complaints-user.md` если есть

## Исходящие артефакты

- Retro-команда (активна — не закрывается до 04-solve)
- `06-retro/02-collect/report-NNN.md` — выжимки всех участников
