# Базовый план: 06-retro-collect

## 1. Определение участников

Прочитай `06-retro/01-recall/report-*.md` и директорию фичи — возьми список участников из recall.

Участники = те кто заполнили `notes-<агент>.md` в recall.

## 2. Создание команды

Создай team через TeamCreate:
- `team_name`: `retro-<последняя часть пути фичи>`
- `description`: `Ретроспектива фичи <feature>`

## 3. Параллельный спаун тиммейтов

Спауни всех участников одновременно с `mode: bypassPermissions`.

Имена subagent_type: `"Танк"`, `"Ада Лавлейс"`, `"Гримм"`, `"Кроули"`, `"Азирафаль"`.

Промпт каждому (подставь имя агента и пути):

> Ты участвуешь в ретроспективе фичи `<feature>` как тиммейт.
>
> Прочитай:
> - `tasks/<feature>/notes-<имя>.md`
> - `tasks/<feature>/complaints-<имя>.md`
> - `tasks/<feature>/notes-user.md` и `tasks/<feature>/complaints-user.md` если существуют
>
> Подготовь выжимку: 3–5 пунктов — что болело сильнее всего, какие паттерны видишь, что предлагаешь изменить.
>
> Отправь выжимку лидеру (SendMessage to team lead). Затем жди вопросов.

## 4. Сбор выжимок

Дождись выжимок от всех тиммейтов.

Собери их в единый список — каждая проблема с атрибуцией агента.

## 5. Формирование отчёта

Запиши `tasks/<feature>/processes/06-retro/02-collect/report-NNN.md`:

```markdown
---
purpose: Collect участников для ретро <feature>
process: 06-retro/02-collect
run: {{N}}
date: {{datetime}}
created: {{datetime}}
status: done
agent: Харли Куин
team: retro-<suffix>
---

## Участники

| Агент | notes (строк) | complaints (строк) |
|---|---|---|
| ... | N | N |

## Выжимки

### <Агент>

- ...

## Итого проблем до группировки: N
```
