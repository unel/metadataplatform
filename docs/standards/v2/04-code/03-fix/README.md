# 04-code/03-fix: Фикс кода

## Что делает

Ада исправляет замечания из code-review. После фикса — возврат к review. Тесты должны оставаться зелёными после каждого фикса.

## Входящие артефакты

- `04-code/02-review/report-NNN.md` — список замечаний с классификацией
- Код реализации

## Исходящие артефакты

- Исправленный код
- `04-code/03-fix/report-NNN.md`

## Навигация

| Условие | Следующий шаг |
|---|---|
| Все замечания исправлены | 04-code/02-review |
| Замечание требует изменения acceptance | `clarification` → 02-acceptance/03-fix |

## Артефакты процесса

- `base-plan.md` — инструкции
- `base-checklist.md` — чек-лист фикса
- `report-NNN.md` — результат (иммутабельный)
