# 02-acceptance: Критерии приёмки

## Что делает группа

Танк пишет acceptance criteria по готовой спеке, Гримм ревьюит на строгость Given/When/Then и измеримость, Танк фиксит. Выходной артефакт: `acceptance.md` прошедший ревью.

## Ключевые правила

- **Given** = только контекст, никаких действий
- **When** = ровно одно действие
- **Then** = наблюдаемый результат с точки зрения вызывающей стороны
- Каждый сценарий независим и измерим

## Процессы

| Процесс | Исполнитель | Что делает |
|---|---|---|
| `01-write` | Танк | Пишет `acceptance.md` по `spec.md` |
| `02-review` | Гримм | Ревьюит строгость GWT, измеримость, anti-patterns |
| `03-fix` | Танк | Фиксит по замечаниям Гримма |

Цикл `02-review → 03-fix` повторяется до `pass`.

## Навигация

| Условие | Следующий шаг |
|---|---|
| `02-review` вернул `pass` | `03-tests/01-write` |
| `02-review` вернул `fail` | `02-acceptance/03-fix` |
