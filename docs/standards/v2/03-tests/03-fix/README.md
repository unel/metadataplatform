# 03-tests/03-fix: Фикс тестов

## Что делает

Исполнитель исправляет конкретные замечания из test-review. Кроули чинит adversarial тесты, Азирафаль — happy path. После фикса — возврат к review.

## Входящие артефакты

- `03-tests/02-review/report-NNN.md` — список замечаний
- Тест-файлы из `tests/happy/` и `tests/adversarial/`

## Исходящие артефакты

- Исправленные тест-файлы
- `03-tests/03-fix/report-NNN.md` — что исправлено, что не исправлено и почему

## Навигация

| Условие | Следующий шаг |
|---|---|
| Все замечания исправлены | 03-tests/02-review |
| Замечание нельзя исправить без изменения acceptance | `clarification` → 02-acceptance/03-fix |

## Артефакты процесса

- `base-plan.md` — инструкции для исполнителей
- `base-checklist.md` — чек-лист фикса
- `report-NNN.md` — результат фикса (иммутабельный)
