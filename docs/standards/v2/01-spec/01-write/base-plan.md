# Базовый план: 01-spec-write

## 1. Сбор контекста

- Прочитай `TASK.md` — что нужно сделать и зачем
- Прочитай `PROJECT.md` — архитектура, стек, ограничения платформы
- Прочитай `PLAN.md` — место фичи в общем roadmap
- Прочитай последний `00-research/01-interview/report-*.md` если существует — что хочет пользователь, зачем, ограничения scope
- Прочитай последний `00-research/02-web/report-*.md` если существует — похожие решения, паттерны, известные failure modes
- Прочитай смежные spec уже реализованных фич — если влияют на эту

## 2. Функциональные требования

- Happy path — что фича делает в штатном сценарии
- Ограничения MVP — что явно не входит в эту итерацию
- Входные данные и ожидаемые результаты — конкретно, с примерами
- Взаимодействие с другими компонентами — что вызывает, что публикует
- Failure modes — все пути ошибок при старте и в работе
- Формат лог-строк — пример для каждого события которое логируется

## 3. Нефункциональные требования

Бери только релевантные категории:

- **Производительность** — время ответа, ожидаемый объём
- **Надёжность** — поведение при сбоях зависимостей, атомарность
- **Параллелизм** — конкурентные запросы, блокировки, идемпотентность
- **Безопасность** — валидация входных данных, ограничения доступа
- **Наблюдаемость** — что логируется (с примером строки), метрики
- **Сопровождаемость** — явные точки расширения, что не должно быть жёстко зашито
- **Ограничения реализации** — зависимости, ограничения платформы

## 4. Проверка полноты

- Пройди по checklist.md — все пункты закрыты?
- При неоднозначности — не выбирай молча, подними вопрос пользователю

## 5. Согласование

- Покажи черновик пользователю
- Жди подтверждения или правок
- После подтверждения — записать report

## Движение вперёд и назад

**Done когда:** черновик подтверждён пользователем, report написан
**Следующий:** 01-spec/02-review
**Передаём:** report-NNN.md (содержит полный текст spec)

**Возврат назад:**
- если входных данных недостаточно → `clarification`, возврат к 00-research/01-interview (через оркестратора)
