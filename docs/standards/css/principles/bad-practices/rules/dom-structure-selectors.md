# Завязка на структуру DOM

Не пиши селекторы которые зависят от конкретной вложенности DOM-элементов.

**Почему:** любое изменение разметки ломает стили. Компонент нельзя переиспользовать в другом контексте. Стили и структура становятся жёстко связаны.

## Плохо

```css
.card div > span > p { color: gray; }
header nav ul li a { text-decoration: none; }
```

## Хорошо

```css
.card-description { color: gray; }
.nav-link { text-decoration: none; }
```
