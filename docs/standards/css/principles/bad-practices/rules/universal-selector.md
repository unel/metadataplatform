# Универсальный селектор (*)

Использование `*` — спорный случай. В одних контекстах оправдано, в других создаёт проблемы производительности.

## Допустимо

```css
/* reset — осознанное применение ко всем элементам */
*, *::before, *::after { box-sizing: border-box; }
```

## Спорно

```css
/* внутри компонента — может быть оправдано */
.card * { color: inherit; }

/* глобально с тяжёлыми свойствами — нагрузка на рендеринг */
* { transition: all 0.3s ease; }
```

## Правило

`*` допустим в reset и изолированных контекстах. Избегай `*` с дорогими свойствами глобально.
