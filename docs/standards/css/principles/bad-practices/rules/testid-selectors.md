# Стили через data-testid

Не используй `data-testid` атрибуты как основу для стилей.

**Почему:** `data-testid` — технический атрибут для тестов, не для стилей. Смешение ответственностей: изменение тестовой разметки ломает стили и наоборот.

## Атрибутные селекторы — с ограничениями

Атрибутные селекторы допустимы, но с ограничениями:

```css
/* Хорошо — семантический атрибут */
[aria-disabled="true"] { opacity: 0.5; }
[data-variant="primary"] { background: var(--color-primary); }

/* Плохо — текст который может измениться или локализоваться */
[placeholder="Введите email"] { ... }
[alt="Логотип компании"] { ... }

/* Плохо — тестовый атрибут */
[data-testid="submit-button"] { ... }
```

## Правило

Атрибутные селекторы допустимы когда атрибут: семантический, стабильный, не локализуемый.
