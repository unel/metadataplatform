# $state

`$state` — реактивное состояние компонента. Используй только для данных которые реально меняются в ответ на действия пользователя или внешние события.

**Когда применять:** данные которые меняются в runtime и должны обновлять UI.

**Когда не применять:** константы; данные которые вычисляются из других — это `$derived`; данные которые приходят снаружи — это `$props`.

## Плохо

```svelte
<script lang="ts">
  let count = $state(0)
  let doubled = $state(0) // вычисляется из count — должен быть $derived

  $effect(() => {
    doubled = count * 2 // синхронизация состояния через $effect — антипаттерн
  })
</script>
```

## Хорошо

```svelte
<script lang="ts">
  let count = $state(0)
  let doubled = $derived(count * 2)
</script>
```

## $state.raw для крупных объектов

Если объект большой и глубокая реактивность не нужна — используй `$state.raw`:

```svelte
<script lang="ts">
  let items = $state.raw<Item[]>([]) // не делает элементы реактивными
</script>
```
