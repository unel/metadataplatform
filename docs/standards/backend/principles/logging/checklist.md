- [ ] Structured logging — JSON формат, не строки
- [ ] Уровни используются по смыслу (DEBUG/INFO/WARN/ERROR)
- [ ] Каждый лог содержит достаточно контекста для диагностики
- [ ] Нет логирования чувствительных данных (пароли, токены, PII)
- [ ] Request ID пробрасывается через весь стек вызовов
- [ ] Ошибки логируются с stack trace
