# Совместимость с Node.js

Bun совместим с большинством Node.js кода, но есть исключения.

## Что работает

- `node:fs`, `node:path`, `node:crypto`, `node:stream` — полная поддержка
- Express, Fastify, большинство HTTP фреймворков
- TypeScript без компиляции — Bun транспилирует на лету

## Что может не работать

- Нативные addon'ы (.node файлы) — не поддерживаются
- Некоторые Worker Threads API — частичная поддержка
- Очень старые CommonJS паттерны

## Как проверять

```bash
# проверить совместимость конкретного пакета
bun add <package> && bun run index.ts

# смотреть на ошибки типа:
# "not implemented in Bun"
# "native addons not supported"
```

При обнаружении несовместимости — ищи Bun-совместимую альтернативу или используй Node.js для этого конкретного сервиса.
