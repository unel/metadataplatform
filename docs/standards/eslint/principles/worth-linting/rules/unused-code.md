# Неиспользуемый код

**Категория:** неиспользуемые переменные, импорты, параметры функций.

**Почему:** неиспользуемый импорт увеличивает бандл. Неиспользуемая переменная — признак незавершённого рефакторинга или баг (забыли использовать результат).

## Примеры правил (не исчерпывающий список)

```json
"no-unused-vars": ["error", { "argsIgnorePattern": "^_" }],
"@typescript-eslint/no-unused-vars": ["error", { "argsIgnorePattern": "^_" }]
```

Префикс `_` — конвенция для намеренно неиспользуемых аргументов (например callback-параметры).
