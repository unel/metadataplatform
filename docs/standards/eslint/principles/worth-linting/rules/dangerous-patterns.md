# Опасные паттерны

**Категория:** паттерны которые почти всегда являются ошибкой или источником багов.

**Примеры категорий:**
- Нестрогое сравнение (`==` вместо `===`) — неявные приведения типов
- Использование `eval` — безопасность и производительность
- Неявный провал в `switch` — забытый `break`
- Необработанные `Promise` — проглоченные ошибки
- Недостижимый код — незавершённый рефакторинг
