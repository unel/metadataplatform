# Форматирование — работа prettier

**Категория:** всё что касается визуального оформления — отступы, длина строки, кавычки, точки с запятой, пробелы.

**Почему:** eslint и prettier конфликтуют на этой территории. Форматирование не является источником багов. Prettier решает это автоматически и без споров.

## Что отключить если есть prettier

Подключи `eslint-config-prettier` — он автоматически отключает все правила которые конфликтуют с prettier:

```json
"extends": ["prettier"]
```

Конкретные правила которые отключаются — на усмотрение prettier. Не перечисляй их вручную.
