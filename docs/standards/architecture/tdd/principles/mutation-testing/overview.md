# Mutation Testing

Способ проверить качество тестов: инструмент автоматически вносит небольшие изменения в код (мутации) и проверяет — упал ли тест. Если мутация выжила (тест не упал) — тест не покрывает это поведение.

Метрика: **mutation score** = убитые мутации / всего мутаций. Высокий score означает что тесты действительно проверяют поведение, а не просто выполняют код.

Инструменты: **Stryker** (TypeScript/SvelteKit), **go-mutesting** или **gremlins** (Go).
