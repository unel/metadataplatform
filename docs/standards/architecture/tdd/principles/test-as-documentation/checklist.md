- [ ] Название теста описывает поведение, не реализацию: «возвращает 404 если entity не найдена», не «тест findEntity»
- [ ] Arrange-Act-Assert разделены визуально — понятно что подготовка, что действие, что проверка
- [ ] Один тест — одно поведение, один assert (или несколько assert одного факта)
- [ ] Тест не знает про внутренние поля и приватные методы объекта
- [ ] Граничные условия покрыты явно: пустой ввод, null, максимальный размер
