# Liskov Substitution Principle

Подтип должен быть полностью заменяем базовым типом. Код использующий базовый тип не должен знать с каким подтипом работает — поведение должно быть предсказуемым.

Признак нарушения: код делает проверку на конкретный подтип (`if instanceof SubType`) чтобы обойти ограниченную реализацию.
