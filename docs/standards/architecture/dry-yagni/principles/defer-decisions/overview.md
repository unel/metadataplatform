# Defer Decisions

Откладывай архитектурные решения до последнего ответственного момента — когда у тебя максимум информации. Принятое слишком рано решение основано на предположениях, а не фактах.

Акцент YAGNI: не строй механизм пока нет реальной потребности — момент когда потребность появилась и есть последний ответственный момент.
