# Pseudocode Programming Process (PPP)

Метод McConnell: перед написанием кода — написать псевдокод на естественном языке, который описывает что делает функция. Рефинировать псевдокод до тех пор пока он не станет прямым планом реализации. Потом превратить в код.

Цель: думать на уровне задачи, а не на уровне синтаксиса. Псевдокод выявляет проблемы дизайна до того как в реализацию вложено время.
