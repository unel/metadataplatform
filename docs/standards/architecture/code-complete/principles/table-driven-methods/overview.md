# Table-Driven Methods

Замена сложной логики (if/else, switch) таблицами данных. Вместо того чтобы кодировать решения в управляющих конструкциях — хранить их в структурах данных и искать по ключу.

McConnell выделяет три типа обращения к таблице: **direct access** (индекс — прямой ключ), **indexed access** (индекс через промежуточную таблицу), **staircase access** (диапазоны значений).

Признак применимости: длинная цепочка if/else или switch где каждая ветка возвращает значение — не выполняет логику.
