- [ ] Модуль — functional или sequential cohesion, не coincidental
- [ ] Нет logical cohesion: функции выбираемые флагом — признак что это разные функции
- [ ] Coupling не выше object-parameter — нет зависимости от внутренней семантики чужого модуля
- [ ] Изменение модуля не вызывает каскад изменений в других — low coupling
- [ ] Всё внутри модуля относится к одной теме — high cohesion
